StrongCrypt-FE

StrongCrypt-FE is a simple and secure single file encryption program for Windows, Linux and Mac OS X. It is a good complement to services such as Dropbox, Google Drive, OneDrive and Box.net.

Features:

  • Open source - no backdoors
  • Support for AES, Serpend and Twofish all with 256 bit strength.
  • Suport for Windows, Linux and Mac OS X.
  • Password gernerator
  • Encrypted files can be renamed the original name is securely stored in the encrypted file
  • Key file Support - any file can be used as a key file
  • PBKDF2 support - iterations can be defined by the user
  • Integrated shredder supporting 5 different shredder methods, including DoD 5220.22 (ECE)
  • Shredding of all temporary and encrypted plaintext files
  • Multi language support for more than 15 languages
  • Shell extension for Windows 32/64 bit
  • Integrated compression
