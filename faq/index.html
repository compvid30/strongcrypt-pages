Frequently Asked Questions

What is StrongCrypt?
StrongCrypt is a on the fly transparent disk encryption program for Windows. Using this software, you can create one or more “virtual disks” on your computer - anything written to these disks is automatically, and securely, encrypted before being stored on your computers hard drive. It is based on FreeOTFE by Sarah Dean.
What is StrongCrypt-FE?
StrongCrypt-FE is a simple and secure single file encryption program for Windows, Linux and Mac OS X. It is a good complement to services such as Dropbox, Google Drive, OneDrive and Box.net. I made this program as a replacement for Ax-Crypt, witch was good for the past. But it does not fit my security requirement anymore.
Is this software open source?
Yes it is. You can view, download or fork the code on Github.
What do I need to compile StrongCrypt?
You need the source code, Delphi 2007 (Delphi 7 may work as well), VisualC++ 2008 or higher, MS Windows SDK and MS Windoes DDK 7600.16385.1.
What do I need to compile StrongCrypt-FE?
You need the source code and Lazarus (FPC 2.6x or higher). You can compile it under Windows, Linux and Mac OS X. Under Linux you need to link it against GTK2. Under Mac OS X you need to link it against Carbon (Don’t try Cocoa, it will not work. Maybe QT works, but I didn’t try).
What does StrongCrypt and StrongCrypt-FE cost?
StrongCrypt and StrongCrypt-FE cost nothing, anyone can use or even modify it under the terms of the licenses. If you have payed for this software something went wrong. If you want to support the project, any donations are appreciated.
