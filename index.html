StrongCrypt is a on the fly transparent disk encryption program for Windows. Using this software, you can create one or more “virtual disks” on your computer - anything written to these disks is automatically, and securely, encrypted before being stored on your computers hard drive. Read more!

StrongCrypt-FE is a simple and secure single file encryption program for Windows, Linux and Mac OS X. It is a good complement to services such as Dropbox, Google Drive, OneDrive and Box.net. Read more!!
